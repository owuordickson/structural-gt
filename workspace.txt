SETTING UP WORKSPACE IN PyCharm

1. SETUP GITHUB
mkdir structural-gt
cd structural-gt/
git init
git remote add origin https://github.com/owuordickson/structural-gt.git
git fetch origin
git fetch origin cppy
git pull origin cppy

2. INSTALL gcc/g++ (C/C++ COMPILER)
Linux/MacOS: use 'brew install gcc' 
Windows: 
download mingw-get-setup.exe installer
run setup As Administrator
install only 'mingw32-gcc-g++'
Add path 'C:\MinGW\bin' to Environment Variables
tesr 'gcc --version'
