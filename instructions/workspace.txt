SETTING UP WORKSPACE IN PyCharm

1. SETUP GITHUB
mkdir structural-gt
cd structural-gt/
git init
git remote add origin https://github.com/owuordickson/structural-gt.git
git fetch origin
git fetch origin cppy
git pull origin main/v3.0.1-terminal


2. SETUP PYCHARM
create a new Project call it "structural-gt"
create another Project "structural-gt-c" and attach it to the previous one
create different pyenv environments for each


3. SETUP Qt DESIGNER
https://www.youtube.com/watch?v=S3qftloOO4U&list=PLScxCGLsaFgpiHgPEScWFiENtpSyx4YNP
pip install pyqt6
pip install pyqt6-tools
open qt-designer (.venv_sgt\Lib\site-packages\qt6_applications\Qt\bin\designer.exe) "Run as administrator"
convert .ui to .py using terminal (pyuic6 -o gui_main_qt.py -x gui.ui)
