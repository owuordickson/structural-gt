BUILD & DISTRIBUTE PRE-COMPILED WHEEL
-------------------------------------

1. Install MinGW and ensure gcc is in your PATH.
    MacOS: brew install gcc
    Windows: download mingw-get-setup.exe installer

2. Ensure you have the required dependencies
    brew install igraph  # macOS (Homebrew)
    sudo apt install libigraph-dev  # Linux (Debian-based)

3. Compile the C extension {this will run without you having to (install in next the step), but with ModuleNotFoundError}
    python setup.py build_ext --inplace  # MacOS/Linux
    python setup.py build_ext --compiler=mingw32 --inplace  # Windows

4. Create a wheel (.whl)
    python -m build --OR-- python setup.py bdist_wheel
    pip install .

5. Install the wheel without needing compilation
    pip install dist/sgt-1.4.5-cp39-cp39-macosx_10_9_x86_64.whl
    or
    pip install sgt-1.4.5-cp39-cp39-win_amd64.whl

CREATING AN EXECUTABLE WITH PyInstaller
---------------------------------------

1. Install Pyinstaller with pip:
    pip install pyinstaller

2. Run PyInstaller once to create the .spec file:
     pyinstaller --windowed --onefile --icon=icon.ico src/StructuralGT.py    # creates StructuralGT.spec

3. Modify the first part of StructuralGT.spec (a) to look like versions in GitHub.

4. Run PyInstaller with .spec file. The executable is in dist/ folder:
    pyinstaller StructuralGT-Win.spec

5a. For Windows, use Inno Setup to create an installer. Share the setup.exe with users for installation and use.

5b. For MacOS, compress the StructuralGT.app file into a zip file. Share the zip file with users.
    Once downloaded by user, unzip the file and drag the app to the Applications folder.
    Users should 'Allow StructuralGT' under Privacy and Security.

5c. For Linux, follow the following steps:
    mkdir sgt_package
    mkdir -p sgt_package/usr/bin
    mkdir -p sgt_package/usr/share/applications
    mkdir -p sgt_package/usr/share/icons/hicolor/128x128/apps
    cp StructuralGT sgt_package/usr/bin
    cp ../icon.png sgt_package/usr/share/icons/hicolor/128x128/apps
    mv sgt_package/usr/share/icons/hicolor/128x128/apps/icon.png sgt_package/usr/share/icons/hicolor/128x128/apps/structuralgt.png
    echo "[Desktop Entry]" > sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Name=StructuralGT" >> sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Exec=/usr/bin/StructuralGT" >> sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Terminal=false" >> sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Type=Application" >> sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Icon=structuralgt" >> sgt_package/usr/share/applications/StructuralGT.desktop
    echo "Categories=Science;Utility;" >> sgt_package/usr/share/applications/StructuralGT.desktop
    mkdir sgt_package/DEBIAN
    echo "Package: structural-gt" > sgt_package/DEBIAN/control
    echo "Version: 3.3.4" >> sgt_package/DEBIAN/control
    echo "Architecture: all" >> sgt_package/DEBIAN/control
    echo "Priority: optional" >> sgt_package/DEBIAN/control
    echo "Section: misc" >> sgt_package/DEBIAN/control
    echo "Maintainer: Dickson Owuor <owuordickson@gmail.com>" >> sgt_package/DEBIAN/control
    echo "Homepage: https://github.com/owuordickson/structural-gt" >> sgt_package/DEBIAN/control
    echo "Description: A software tool for graph theory analysis of microscopy images." >> sgt_package/DEBIAN/control
    dpkg-deb --build sgt_package

    Install via:
    cd ~/Downloads
    sudo dpkg -i sgt_package.deb

6. For the terminal executable,
    Open a terminal application and execute the following command:
    StructuralGT-cli -f datasets/InVitroBioFilm.png -c datasets/sgt_configs.ini -t 1
