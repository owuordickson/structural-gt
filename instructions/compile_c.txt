SETTING UP WORKSPACE IN PyCharm

1. SETUP GITHUB
    mkdir structural-gt
    cd structural-gt/
    git init
    git remote add origin https://github.com/owuordickson/structural-gt.git
    git fetch origin
    git fetch origin cppy
    git pull origin cppy

2. INSTALL gcc/g++ (C/C++ COMPILER)
Linux/MacOS:
    use 'brew install gcc'
Windows (Go with Option 2):
    -- Option 1 --
    download and install 'Microsoft C++ Build Tools' (install Desktop development with C++)
    * Use 'Launch' and command where.exe to find: clang/cl and cmake and nmake
    -- Option 2 --
    https://www.mingw-w64.org/getting-started/msys2/
    * Install MSYS2(GCC) Software management library (it is better than MinGW).
    * Install location is usually in "C:\msys64"
    * Search for packages here: https://packages.msys2.org/queue
    * Open "MSYS2 UCRT64" as Admin and install gcc:
        pacman -S mingw-w64-ucrt-x86_64-gcc
    * After installation test:
        gcc --version
        where.exe gcc

3. INSTALL CMAKE (for Windows, ignore for MacOS)
Linux/MacOS:
    use 'brew install cmake'
Windows (Go with Option 2):
    -- Option 1 --
    use 'Microsoft C++ Build Tools' to install
    -- Option 2 --
    pacman -S cmake
    test 'cmake --version'

4. C LIBRARIES (iGraph pre-requisites)
4a. PTHREADS library
    Linux/macOS:
        Already installed
    Windows (use Option 2):
        -- Option 1 --
        Install Microsoft Build Tools
        We create the library files using 'nmake clean VC-small-static' through 'Developer Command Prompt' (choose x86/x64)
        Copy header files and .dll and .lib files to MinGW folder
        -- Option 2 --
        * Open "MSYS2 UCRT64" as Admin and install:
            pacman -S mingw-w64-ucrt-x86_64-libwinpthread-git
4b. BISON/Flex
    Linux/macOS:
        Already installed
    Windows:
        -- Option 2 --
        * Open "MSYS2 UCRT64" as Admin and install:
            pacman -S bison
            pacman -S flex

4b. IGRAPH library
    Linux/macOS:
        brew install igraph
    Windows:
    -- Option 1 --
        * Pre-requisites
        * Install Microsoft Build Tools
        * https://igraph.org/c/html/latest/igraph-Installation.html#igraph-Installation-installation
        * Use the 'Developer Command Prompt' to run the following commands (choose x86/x64):
            mkdir build
            cd build
            cmake ..
            cmake --build . --config Release
            cmake --build . --target check --config Release
            Copy (include) header files and igraph.lib files to MinGW folder
    -- Option 2 --
        * Open "MSYS2 UCRT64" as Admin and install:
            pacman -S mingw-w64-ucrt-x86_64-igraph

5. COMPILING in WINDOWS
clang .\test.c -o .\test.exe
gcc .\test.c -o .\test

Windows:
gcc .\igraph_test.c -I../SGT-c -IC:/MinGW/include/igraph -IC:/MinGW/include/pthread -LC:/MinGW/lib/igraph -LC:/MinGW/lib/igraph -ligraph -lpthread -o .\igraph_test
gcc .\sgt_test.c -I../SGT-c -IC:/MinGW/include/igraph -IC:/MinGW/include/pthread -LC:/MinGW/lib/igraph -LC:/MinGW/lib/igraph -ligraph -lpthread -o .\sgt_test
